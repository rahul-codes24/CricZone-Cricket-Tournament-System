# KPL Tournament Website - .NET 10 MVC

## What is included
- Home dashboard
- Points table with net run rate
- Teams and players
- Match result, toss and scorecard
- Batting score rows
- Bowling score rows
- Tournament stats: orange cap, purple cap, most sixes
- Admin-only login for changes
- HTML + CSS custom UI (no drag and drop)
- SQLite database

## Default admin login
- Username: admin
- Password: admin123

## How to run
1. Install .NET 10 SDK.
2. Open terminal in this project folder.
3. Run: `dotnet restore`
4. Run: `dotnet run`
5. Open the URL shown in terminal.

## Best tool to use
- Visual Studio 2026 or later for full .NET 10 support.
- You can also run with the .NET CLI.

## Important note
This is a starter project. You can expand it with:
- player replacement history
- season awards table
- best fielder entry screen
- knockout/final stage
- photo upload for teams and players
- ball-by-ball scoring
